# Dark Eden RPG Game

## Overview
Dark Eden is a text-based RPG game developed in Java, where players create characters, equip gear, and engage in randomized combat. The game is designed with a focus on item management and probabilistic events to make each playthrough unique.

## Features
- Player customization with gear and weapons
- Randomized combat events
- File-based inventory system using text files for gear, items, and weapons
- Simplified object-oriented code for easy updates

## Files and Structure

- `Main.java`: The main class responsible for initializing the game and managing core gameplay logic.
- `Play.java`: Handles the combat mechanics and player interactions during the game.
- `Action.txt`: Stores different actions players can take, such as attacking or defending during combat.
- `Gear.txt`: Stores the playerâ€™s equippable armor and gear.
- `item.txt`: Contains various items available for the player, such as potions.
- `wpn.txt`: Defines available weapons and their stats.

## How to Play
1. Run the `Main.java` file to start the game.
2. Create your character by selecting gear and weapons.
3. Engage in combat, use items, and explore the world.
4. Win by defeating enemies through strategic gameplay and using your inventory.
